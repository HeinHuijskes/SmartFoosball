The smart foosball table

This website is connected to the smart foosball table. On the smart foosball table a match of table foosball can be played as on a normal foosball table. However, this foosball table is made smart and can collect and display data on the foosball match. The smart foosball table displays a camera feed with ball tracking on this website, as well as the average ball speed taken from this. In addition, with distance sensors the foosball table is able to detect goals being scored and display the score on the website.

Starting a match

To start a match using the smart foosball table, the system has to be running. The system is started if the website is running. The 'new game' button can be found on the 'live game' page, which can be navigated to by pressing 'watch match' on the starting screen of the website. This 'new game' button can be used to reset the game scores and start a new game.

How it works

When playing on this table, the score is kept by the smart foosball table. Distance sensors in both of the goals of the foosball table are installed to detect movement. This movement in the goal is registered as a scored point. The smart foosball table keeps track of the score of a match and displays this on the website. On the website the score as well as a live camera feed and the average speed of the ball are displayed. The camera is place directly above the table and the codes on all four corners of the foosball table are used to detect the foosball field. With the use of these codes, called the ArUco markers, the video captured by the camera can be cropped and tilted to display to foosball field on the website. The smart foosball table uses the camera footage to compile data, the system can use the footage to track the ball in the field and to compile the average speed.

De-assembling the smart foosball table

The smart foosball table consists of wooden frame which holds the camera, this can be taken off. The wooden frame consists of three components. These three components are connected with threaded inserts which can be unscrewed and re-screwed. The first component is the wooden plank that holds the camera, this can be unscrewed and taken off. (This consists of unscrewing two threaded inserts.) Then two components remain which are no longer connected, but are similar in structure. The structure has two threaded inserts on either side of the table, four per component. After unscrewing all 8 threaded inserts the second and third component can also be removed. Note that while the structure of the second and third component are similar, the components can only be placed on one side of the foosball table and will not fit on the other side.
